Como usar

Inclua o token no header de cada requisição:

Authorization: Bearer bc_seu_token_aqui

1. Listar grupos (provedores/clientes)

GET /api/v1/external/groups

2. Listar backups de um grupo

GET /api/v1/external/groups/{GROUP_ID}/backups?status=success

Retorna a listagem contendo o "id" do backup e, a partir de agora, o "device_id" além do "device_name".

3. Download do arquivo de backup

GET /api/v1/external/backups/{BACKUP_ID}/download

Documentação completa disponível em /docs (Swagger UI).
